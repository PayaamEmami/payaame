 Variables

Rules for variable names

  • Can't start with a number
  • Can't use any of these symbols: : " ' < > / ? | \ ( ) ! @ # $ % ^ & * ` + -
  • Can't use spaces
 Strings

Strings are sequences of characters, using the syntax of either single quotes or double quotes.

  • 'hello'
  • "Hello"
  • "I don't do that"

Indexing and Slicing

Indexing is used with [] and the index number to indicate the position (starting with 0).

Slicing allows you to grab a subsection of multiple characters, a "slice" of the string. The syntax of slicing follows this format: [start:stop:step]. Step is the size of the number of indexes to jump through each iteration.

print() and len()

The print() function displays the message to the screen.

The len() function returns the number of items of an object.

Formatting strings

There is more than one way to format a string with variables:

  • print('Hello, my name is {{ "{" }}{{ "}" }}.format(name))
  • print(f'Hello, my name is {{ "{" }}name{{ "}" }}')
 Lists

Lists are ordered sequences that can hold a variety of object types.

They use [ ] brackets and commas to separate objects in the list.

  • [1,2,3,4,5]

Lists support indexing and slicing. Lists can be nested and also have a variety of useful methods that can be called off of them (sort, pop, append, etc).

Dictionaries

Dictionaries are unordered mappings for storing objects.

Dictionaries uses key-value pairing in comparison to lists that store objects in an ordered sequence. This key-value pair allows users to quickly grab objects without needing to know an index location.

Dictionaries use curly braces and colons to signify the keys and their associated values: {{ "{" }} 'key1':'value1', 'key2':'value2', 'key3:'value3' {{ "}" }}

Dictionaries vs Lists

Dictionaries are objects retrieved by key name. They are unordered, and can't be sorted.

Lists are objects retrieved by location. They are ordered, and can be indexed, sliced, and sorted.

Tuples

Tuples are very similar to lists. The one key difference: immutability.

Once an element is inside a tuple, it can't be reassigned.

Tuples use parenthesis: (1,2,3)

Sets

Sets are unordered collections of unique elements (there can only be one representative of the same object).

myset = set()
myset.add(1)
# The value of the set is: {{ "{" }} 1 {{ "}" }}

Conditional Statements

Conditional statements use colons and indentation (whitespace).

If, Elif, Else

if some_condition:
# execute some code
elif some_other_condition:
# execute some other code
else:
# execute something else

For Loops

Many objects in Python are iterable, meaning we can iterate over every element in the object. We can use for loops to execute a block of code for every iteration.

for item_name in my_iterable_object:
print(item_name)

While Loops

While loops will continue to execute a black of code while some condition remains true.

while some_boolean_condition:
# execute some code
else:
# execute some other code
