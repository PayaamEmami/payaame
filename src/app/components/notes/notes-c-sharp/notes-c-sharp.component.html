Strings

String Interpolation

The $ character identifies a string literal as an interpolated string.

  • Literal string: "I like to eat pizza"
  • Interpolated string: $"I like to eat {{'{'}}food{{'}'}}"

Whitespace

Use the Trim() method to remove unwanted leading and trailing whitespace.

  • TrimStart() can be used for leading whitespace and TrimEnd() can be used for trailing whitespace

Replace

Replace() can be used to replace substrings within a given string.

  • food.Replace("pizza", "pasta")

Capitalization

ToUpper() and ToLower() are useful methods for making a string ALL CAPS or all lower case.

  • food.ToUpper() // PIZZA
  • food.ToLower() // pizza

Note: these methods return a copy and won't change the original value in the string.

Searching strings

The Contains() method returns a boolean value which tells you if the string you were searching for was found.

  • songLyrics.Contains("Scaramouche, will you do the Fandango?")

StartsWith() and EndsWith() search for substrings at the beginning and ending respectively.

Numbers and Integer Math

There are more number types than I am going to list here however for most cases int will usually do what is required. For more complex programs that may require large or precise numbers, it is important to take a look at all the different types and compare the range for each type as well as the memory requirements.

Integer

The int type represents an integer, a positive or negative whole number.

Common arithmetic operations for integers in C# include:

  • + for addition
  • - for subtraction
  • * for multiplication
  • / for division
  • % for modulo
  • ++ for increment
  • -- for decrement

Order of operations in mathematics will work the same in C# (PEMDAS).

Integer division will always produce an integer result.

  • 11 / 3 = 3

Integer precision and limits

Integer division will truncate the result, so the modulo operator can be used to retrieve the remainder.

In mathematics, numbers can go infinitely, however in C# there is a minimum and maximum number that is limited to how much the computer can keep in memory.

  • int.MaxValue and int.MinValue will retrieve these minimum and maximum values

Integer overflow will 'flip the odometer' and wrap around to negative integers.

Doubles

Doubles are floating-point types that let you work with even larger numbers.

Doubles also allow you to work with decimal point values.

Decimals

The decimal type has a smaller range but greater precision than double.
