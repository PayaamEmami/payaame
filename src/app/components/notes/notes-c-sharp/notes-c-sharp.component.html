Strings

String Interpolation

The $ character identifies a string literal as an interpolated string.

  • Literal string: "I like to eat pizza"
  • Interpolated string: $"I like to eat {{'{'}}food{{'}'}}"

Whitespace

Use the Trim() method to remove unwanted leading and trailing whitespace.

  • TrimStart() can be used for leading whitespace and TrimEnd() can be used for trailing whitespace

Replace

Replace() can be used to replace substrings within a given string.

  • food.Replace("pizza", "pasta")

Capitalization

ToUpper() and ToLower() are useful methods for making a string ALL CAPS or all lower case.

  • food.ToUpper() // PIZZA
  • food.ToLower() // pizza

Note: these methods return a copy and won't change the original value in the string.

Searching strings

The Contains() method returns a boolean value which tells you if the string you were searching for was found.

  • songLyrics.Contains("Scaramouche, will you do the Fandango?")

StartsWith() and EndsWith() search for substrings at the beginning and ending respectively.
