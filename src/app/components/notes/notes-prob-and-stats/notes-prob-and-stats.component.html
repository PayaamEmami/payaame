Visualizing Data

One-way data

One-way data, in comparison to two-way data, contains a single individual, that has properties (variables) attached to it.

Individuals are items in a data set and can be cases, things, people, etc.

Variables are properties that describe individuals. Variables can be categorical or quantitative:

  • Categorical variables are non-numerical values
  • Quantitative variables are numerical values

Together, individuals and variables are called data.

Bar graphs and pie charts

Bar graphs are best for representing data that isn't connected, more categorical data.

For horizontal bar graphs, it's common to put in descending order.

Pie charts are best for when comparing parts of a whole.

Line graphs and ogives

Line graphs are best for representing change over time, so we can see how a variable moves up or down, and we can see the rate of change between two data points.

Ogives are best for when you want to show an accumulated total, you can see rate of change between each data point as well as the accumulated total over time.

Two-way data

Data that is dependent on more than one individual.

We talk about two-way data in terms of independent and dependent variables.

  • Independent variables are individuals such as cases, things, people, etc
  • Dependent variables are variables that describe independent variables

In one-way data, we have one independent variable (individual) and one or more dependent variables (variables).

Venn diagrams

Two-way data can be represented as a venn diagram.

Frequency tables and dot plots

Frequency tables display how frequently or infrequently something occurs.

Dot plots can be used to display frequency of small data sets.

Relative frequency tables

Relative frequency tables are tables that display percentages instead of actual counts.


Column-relative frequency and row-relative frequency are tables in which the summation of the total column or total row equates to 1.

Total relative frequency tables are calculated for a two-way table by dividing every cell by the grand total.

Joint distributions

Joint distribution tables contain percentages comparing one data set to another.

Marginal distribution is the total column or total row in a joint distribution.

Conditional distribution is the distribution of one variable, given the particular value of another variable.

Histograms and stem-and-leaf plots

A histogram, AKA a frequency histogram, is just like a bar graph, except we collect the data into buckets and then sketch a bar for each bucket.

The way the data is spread out is called a distribution.
As a general rule, qualitative data is usually better for a bar graph and quantitative data is usually better in a histogram.

A stem-and-leaf plot, AKA stem plot, is just another way to summarize data.

Box-and-whisker plots

Box-and-whisker plots (also called box plots) are a great way to represent a data set when you want to show the median and spread of the data at the same time.

The five-number summary, also called the five-figure summary, for any set of data will include the minimum and maximum values, the median, and Q1 and Q3 for the data set. We usually give the five-number summary in a table, and we can easily gather all of this information from a box-plot.

Analyzing Data

Measures of central tendency

Measures of central tendency are different ways we've come up with to describe the middle, center, or most typical value of the data.

  • Mean

    • We usually say "average" but technically we're thinking about mean, also called the arithmetic mean. The formula can be simplified to the sum of all the data points divided by the number of data points. We usually denote the mean with the greek symbol μ (pronounced "mew").

  • Median

    • The median of a data set is the value we get when we line up all the data points in the set from least to greatest, and then we look at the number or pair of numbers in the middle.

  • Mode

    • The mode of a data set is the value that occurs most often, more than any other value. You could think about it as the most typical value in the set. If a data set has two modes (a tie between two values that occur the most), we call the data set bi-modal. Note, others would say the data set has no mode, instead of bi-modal.

Measures of dispersion

Measures of dispersion or scatter describe how and by how much our data set is spread out around its center.

  • Range

    • The range of a data set is the difference between the largest value and the smallest value.

  • Interquartile range (IQR)

    • The interquartile range is the difference between the median of the upper half and the median of the lower half (of the data set).

Outlier

A number on the extreme upper end or extreme lower end of a data set.
